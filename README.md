Ansible Role: MySQL charset
=========

Setting MySQL character-set on any Linux system. 

Requirements
------------

`mysql` should be installed.

Role Variables
--------------

None.

Dependencies
------------

None.

Example Playbook
----------------

    - hosts: servers
      roles:
         - k-miyabe.mysql-charset
